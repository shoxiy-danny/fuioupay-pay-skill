# 3.3 公众号/服务窗/小程序统一下单

## 请求地址

| 环境 | 地址 |
|------|------|
| 测试 | `https://fundwx.payfuiouo2o.com/preCreate` |
| 生产 | `https://spay-cloud.fuioupay.com/preCreate` 或 `https://spay-cloud2.fuioupay.com/preCreate` |

## 订单类型

| order_type | 说明 |
|------------|------|
| JSAPI | 微信公众号支付 |
| FWC | 支付宝服务窗/小程序 |
| LETPAY | 微信小程序 |
| MPAY | 云闪付小程序 |

## 请求报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | version | 是 | String | 1.0 |
| 2 | ins_cd | 是 | String | 机构号 |
| 3 | mchnt_cd | 是 | String | 商户号 |
| 4 | term_id | 是 | String | 终端号 |
| 5 | random_str | 是 | String | 随机字符串 |
| 6 | sign | 是 | String | 签名 |
| 7 | order_type | 是 | String | 订单类型 |
| 8 | mchnt_order_no | 是 | String | 商户订单号 |
| 9 | order_amt | 是 | Number | 订单金额，单位：分 |
| 10 | goods_des | 是 | String | 商品描述 |
| 11 | txn_begin_ts | 是 | String | 交易时间yyyyMMddHHmmss |
| 12 | notify_url | 是 | String | 回调地址 |
| 13 | term_ip | 是 | String | 终端IP |
| 14 | sub_appid | 条件必填 | String | 子商户公众号ID（小程序必填）|
| 15 | sub_openid | 条件必填 | String | 用户openid（JSAPI必填）|
| 16 | trade_type | 是 | String | 交易类型 |

## 响应报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | result_code | 是 | String | 错误代码 |
| 2 | result_msg | 是 | String | 错误描述 |
| 3 | qr_code | 是 | String | 二维码链接（用于H5调起）|
| 4 | sdk_appid | 是 | String | SDK用appid |
| 5 | sdk_noncestr | 是 | String | SDK用nonce |
| 6 | sdk_package | 是 | String | SDK用package |
| 7 | sdk_signtype | 是 | String | SDK用签名类型 |
| 8 | sdk_timestamp | 是 | String | SDK用时间戳 |
| 9 | sdk_paysign | 是 | String | SDK用签名 |
| 10 | session_id | 是 | String | APP支付用session_id |

## 特别说明

### 微信JSAPI

1. 入网商户名称必须跟公众号主体名称一致
2. 需要配置AppSecret
3. 获取openid建议采取静默模式

### 支付宝服务窗

1. 需要获取buyer_id（user_id）
2. 参考：https://doc.open.alipay.com/docs/doc.htm?spm=a219a.7629140.0.0.cPv1DX&treeId=193&articleId=105193&docType=1#s10

### 银联云闪付JS调起支付

使用响应中的qr_code字段调起支付。
