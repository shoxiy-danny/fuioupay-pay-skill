# 3.8 撤销接口

## 请求地址

| 环境 | 地址 |
|------|------|
| 测试 | `https://fundwx.payfuiouo2o.com/cancelorder` |
| 生产 | `https://spay-cloud.fuioupay.com/cancelorder` 或 `https://spay-cloud2.fuioupay.com/cancelorder` |

## 场景说明

本接口只支持**条码支付撤销**。

**建议**: 先发起查询确认订单状态未支付（或者支付中）后发起撤销。如果富友测原交易状态成功，该接口功能同退款接口，但还是建议使用退款接口。

**注意**: 成功交易调用此接口会退款，请谨慎调用。

**建议**: 原交易请求完至少15秒后发起撤销。

**目前仅支持撤销当日交易。**

## 请求报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | version | 是 | String | 1.0 |
| 2 | ins_cd | 是 | String | 机构号 |
| 3 | mchnt_cd | 是 | String | 商户号 |
| 4 | term_id | 是 | String | 终端号 |
| 5 | mchnt_order_no | 是 | String | 商户订单号 |
| 6 | random_str | 是 | String | 随机字符串 |
| 7 | sign | 是 | String | 签名 |
| 8 | order_type | 是 | String | 订单类型：ALIPAY/WECHAT/UNIONPAY/JDBT |
| 9 | cancel_order_no | 是 | String | 商户撤销单号 |
| 10 | operator_id | 否 | String | 操作员 |
| 11 | reserved_fy_term_id | 否 | String | 富友终端号 |

## 响应报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | result_code | 是 | String | 错误代码，000000成功 |
| 2 | result_msg | 是 | String | 错误描述 |
| 3 | ins_cd | 是 | String | 机构号 |
| 4 | mchnt_cd | 是 | String | 商户号 |
| 5 | term_id | 否 | String | 终端号 |
| 6 | random_str | 是 | String | 随机字符串 |
| 7 | sign | 是 | String | 签名 |
| 8 | order_type | 是 | String | 订单类型 |
| 9 | mchnt_order_no | 是 | String | 原交易商户订单号 |
| 10 | cancel_order_no | 是 | String | 商户撤销单号 |
| 11 | transaction_id | 是 | String | 渠道交易流水号 |
| 12 | cancel_id | 否 | String | 渠道撤销流水号 |
| 13 | fund_change | 否 | String | 是否有资金流向（仅支付宝有）|
| 14 | recall | 否 | String | 是否需要重新调用撤销（Y-需要）|
| 15 | reserved_fy_settle_dt | 是 | String | 富友交易日期 |

## 代码示例

```javascript
// 撤销订单
const result = await payClient.reverse({
  mchntCd: '000XXXXXXX',
  orderType: 'WECHAT',
  mchntOrderNo: '20240101120000001',
  cancelOrderNo: '20240101120000001_cancel'
});

if (result.result_code === '000000') {
  console.log('撤销成功');
  if (result.recall === 'Y') {
    // 需要重新调用撤销
  }
}
```
