# 3.6 退款申请

## 请求地址

| 环境 | 地址 |
|------|------|
| 测试 | `https://fundwx.payfuiouo2o.com/commonRefund` |
| 生产 | `https://spay-cloud.fuioupay.com/commonRefund` 或 `https://spay-cloud2.fuioupay.com/commonRefund` |

## 场景说明

当交易发生之后，由于买家或者卖家的原因需要退款时，卖家可以通过退款接口将支付款退还给买家。

- **总退款金额不能超过订单总金额**
- **退款只有当商户账户有资金才能退款**
- 微信360天、支付宝300天、银联180天支持全额或多次部分退款
- 如需支持360天内退款，需填写 `reserved_origi_dt`（原交易日期）

## 请求报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | version | 是 | String | 1.0 |
| 2 | ins_cd | 是 | String | 机构号 |
| 3 | mchnt_cd | 是 | String | 商户号 |
| 4 | term_id | 是 | String | 终端号 |
| 5 | mchnt_order_no | 是 | String | 原商户订单号 |
| 6 | random_str | 是 | String | 随机字符串 |
| 7 | sign | 是 | String | 签名 |
| 8 | order_type | 是 | String | 订单类型 |
| 9 | refund_order_no | 是 | String | 商户退款单号 |
| 10 | total_amt | 是 | Number | 总金额（分）|
| 11 | refund_amt | 是 | Number | 退款金额（分）|
| 12 | operator_id | 否 | String | 操作员 |
| 13 | reserved_origi_dt | 否 | String | 原交易日期yyyyMMdd，不填支持30天内退款 |

| 15 | reserved_refund_desc | 否 | String | 退款原因（微信支付、支付宝PC端展示）|

## 退款单号规则

- **5-30字符**
- **只能包含字母、数字、下划线**
- **区分大小写**

## 代码示例

```javascript
// 生成退款单号
function generateRefundOrderNo(origOrderNo) {
  const random2 = Math.floor(Math.random() * 100).toString().padStart(2, '0');
  const maxOrigLen = 30 - 3; // 预留 "_R" + 2位数字
  const truncated = origOrderNo.length > maxOrigLen
    ? origOrderNo.substring(0, maxOrigLen)
    : origOrderNo;
  return `${truncated}_R${random2}`;
}

// 退款申请
const result = await payClient.refund({
  mchntCd: '000XXXXXXX',
  orderType: 'WECHAT',
  mchntOrderNo: '20240101120000001',
  refundOrderNo: generateRefundOrderNo('20240101120000001'),
  totalAmt: 100,      // 分
  refundAmt: 50       // 分
});

if (result.result_code === '000000') {
  console.log('退款成功:', result.refund_id);
}
```

## 注意事项

1. **验签**: 富友返回的sign可能含换行符，需要 `.replace(/\s/g, '')`
2. **退款单号**: 必须符合5-30字符、字母数字下划线规则
3. **部分退款**: 支持多次部分退款，累计不超过订单总金额
