# 3.7 关闭订单

## 请求地址

| 环境 | 地址 |
|------|------|
| 测试 | `https://fundwx.payfuiouo2o.com/closeorder` |
| 生产 | `https://spay-cloud.fuioupay.com/closeorder` 或 `https://spay-cloud2.fuioupay.com/closeorder` |

## 场景说明

本接口支持微信、支付宝各种下单类交易的关闭。**只有未支付的订单才能发起关闭。**

### 支付宝主扫

如果未扫码直接关单会失败，一定要扫码后未支付才能调用关单接口。支付宝动态码的交易，下单成功（返回动态码链接只是预下单），因此在查询、关单的时候支付宝会返回"交易不存在"，用户扫码以后才会去下单。

### 微信主扫

调用了统一下单接口后，就可以直接关单，不需要去扫qr_code产生的二维码链接。

## 请求报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | version | 是 | String | 1.0 |
| 2 | ins_cd | 是 | String | 机构号 |
| 3 | mchnt_cd | 是 | String | 商户号 |
| 4 | term_id | 是 | String | 终端号 |
| 5 | mchnt_order_no | 是 | String | 商户订单号 |
| 6 | random_str | 是 | String | 随机字符串 |
| 7 | sign | 是 | String | 签名 |
| 8 | order_type | 是 | String | 订单类型：ALIPAY/WECHAT/UNIONPAY |
| 9 | sub_appid | 否 | String | 子商户公众号ID |

## 响应报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | result_code | 是 | String | 错误代码，000000成功 |
| 2 | result_msg | 是 | String | 错误描述 |
| 3 | ins_cd | 是 | String | 机构号 |
| 4 | mchnt_cd | 是 | String | 商户号 |
| 5 | term_id | 否 | String | 终端号 |
| 6 | random_str | 是 | String | 随机字符串 |
| 7 | sign | 是 | String | 签名 |
| 8 | order_type | 是 | String | 订单类型 |
| 9 | mchnt_order_no | 是 | String | 商户订单号 |

## 代码示例

```javascript
// 关闭订单
const result = await payClient.closeOrder({
  mchntCd: '000XXXXXXX',
  orderType: 'WECHAT',
  mchntOrderNo: '20240101120000001'
});

if (result.result_code === '000000') {
  console.log('订单已关闭');
}
```
