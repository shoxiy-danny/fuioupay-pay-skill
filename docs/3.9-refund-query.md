# 3.9 退款查询

## 请求地址

| 环境 | 地址 |
|------|------|
| 测试 | `https://fundwx.payfuiouo2o.com/refundQuery` |
| 生产 | `https://spay-cloud.fuioupay.com/refundQuery` 或 `https://spay-cloud2.fuioupay.com/refundQuery` |

## 场景说明

本接口支持所有退款交易查询，**时间限制：3日内**。

## 请求报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | version | 是 | String | 1.0 |
| 2 | ins_cd | 是 | String | 机构号 |
| 3 | mchnt_cd | 是 | String | 商户号 |
| 4 | term_id | 是 | String | 终端号 |
| 5 | refund_order_no | 是 | String | 商户退款单号 |
| 6 | random_str | 是 | String | 随机字符串 |
| 7 | sign | 是 | String | 签名 |

## 响应报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | result_code | 是 | String | 错误代码，000000成功 |
| 2 | result_msg | 是 | String | 错误描述 |
| 3 | trans_stat | 是 | String | 交易状态 |
| 4 | ins_cd | 是 | String | 机构号 |
| 5 | mchnt_cd | 是 | String | 商户号 |
| 6 | term_id | 否 | String | 终端号 |
| 7 | random_str | 是 | String | 随机字符串 |
| 8 | sign | 是 | String | 签名 |
| 9 | order_type | 是 | String | 订单类型 |
| 10 | mchnt_order_no | 是 | String | 商户订单号 |
| 11 | refund_order_no | 是 | String | 商户退款单号 |
| 12 | transaction_id | 否 | String | 渠道交易流水号 |
| 13 | refund_id | 否 | String | 渠道退款流水号 |
| 14 | reserved_fy_settle_dt | 是 | String | 富友交易日期 |
| 15 | reserved_fy_trace_no | 否 | String | 富友系统内部追踪号 |
| 16 | reserved_refund_amt | 否 | Number | 退款金额 |

## 退款状态 (trans_stat)

| 状态 | 说明 |
|------|------|
| SUCCESS | 退款成功 |
| PAYERROR | 退款失败 |

## 代码示例

```javascript
// 查询退款
const result = await payClient.queryRefund({
  mchntCd: '000XXXXXXX',
  refundOrderNo: '20240101120000001_R01'
});

console.log('退款状态:', result.trans_stat);
```
