# 3.4 订单查询

## 请求地址

| 环境 | 地址 |
|------|------|
| 测试 | `https://fundwx.payfuiouo2o.com/commonQuery` |
| 生产 | `https://spay-cloud.fuioupay.com/commonQuery` 或 `https://spay-cloud2.fuioupay.com/commonQuery` |

## 场景说明

由于不可抗因素导致交易超时，或者状态不明确时，需要商户主动发起交易查询来确定交易的最终状态。**目前系统支持往前查询3日的交易**。

## 请求报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | version | 是 | String | 1.0 |
| 2 | ins_cd | 是 | String | 机构号 |
| 3 | mchnt_cd | 是 | String | 商户号 |
| 4 | term_id | 是 | String | 终端号 |
| 5 | order_type | 是 | String | 订单类型 |
| 6 | mchnt_order_no | 是 | String | 商户订单号 |
| 7 | random_str | 是 | String | 随机字符串 |
| 8 | sign | 是 | String | 签名 |

## 订单类型说明

| order_type | 说明 |
|------------|------|
| ALIPAY | 支付宝 |
| WECHAT | 微信 |
| UNIONPAY | 银联二维码 |
| BESTPAY | 翼支付 |
| JSAPI | 公众号支付 |
| FWC | 支付宝服务窗/小程序 |
| LETPAY | 微信小程序 |
| MPAY | 云闪付小程序 |
| PY68/PY69 | 银联分期 |
| JDBT | 京东白条 |

## 响应报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | result_code | 是 | String | 错误代码，000000成功 |
| 2 | result_msg | 是 | String | 错误描述 |
| 3 | ins_cd | 是 | String | 机构号 |
| 4 | mchnt_cd | 是 | String | 商户号 |
| 5 | random_str | 是 | String | 随机字符串 |
| 6 | sign | 是 | String | 签名 |
| 7 | trans_stat | 是 | String | 查询状态 |
| 8 | order_amt | 是 | Number | 订单金额，单位：分 |
| 9 | transaction_id | 是 | String | 渠道订单号 |
| 10 | mchnt_order_no | 是 | String | 商户订单号 |
| 11 | buyer_id | 否 | String | 买家在渠道的用户ID |
| 12 | reserved_fy_settle_dt | 是 | String | 富友交易日期 |
| 13 | reserved_coupon_fee | 否 | String | 优惠金额（分）|
| 14 | reserved_is_credit | 否 | String | 1-信用卡/花呗，0-其他 |
| 15 | reserved_txn_fin_ts | 否 | String | 用户支付时间 |
| 16 | reserved_bank_type | 否 | String | 付款方式 |

## 交易状态 (trans_stat)

| 状态 | 说明 |
|------|------|
| SUCCESS / TRADE_SUCCESS | 交易成功 |
| FAIL / TRADE_CLOSED | 交易失败/已关闭 |
| NOTPAY / WAIT_BUYER_PAY | 未支付 |
| USERPAYING | 用户支付中 |
| REFUND | 已退款 |

## 代码示例

```javascript
// 查询订单
const result = await payClient.queryOrder({
  mchntCd: '000XXXXXXX',
  orderType: 'WECHAT',
  mchntOrderNo: '20240101120000001'
});

console.log('状态:', result.trans_stat);
console.log('订单金额:', result.order_amt);
console.log('渠道订单号:', result.transaction_id);
```
