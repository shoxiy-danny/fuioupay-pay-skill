# 3.14 微信委托代扣查询

## 请求地址

- 测试地址：`https://fundwx.payfuiouo2o.com/withholdQuery`
- 生产地址：待富友提供

## 场景说明

该接口提供所有微信支付订单的查询，商户可以通过该接口主动查询订单状态，完成下一步的业务逻辑。

需要调用查询接口的情况：
1) 当商户后台、网络、服务器等出现异常，商户系统最终未接收到支付通知；
2) 调用支付接口后，返回系统错误或未知交易状态情况；
3) 调用代扣支付API，返回ACCEPTED、NOPAY或USERPAYING的状态。

## 微信委托代扣查询请求报文

| 序号 | 参数名 | 出现要求 | 型态 | 长度 | 说明 |
|------|--------|----------|------|------|------|
| 01 | version | 必填 | String | 8 | 1.0 |
| 02 | ins_cd | 必填 | String | 20 | 机构号,接入机构在富友的唯一代码 |
| 03 | mchnt_cd | 必填 | String | 15 | 商户号,富友分配给二级商户的商户号 |
| 04 | term_id | 必填 | String | 8 | 终端号(没有真实终端号统一填88888888) |
| 05 | order_type | 必填 | String | 20 | 订单类型: WXPAP(微信委托代扣) |
| 06 | mchnt_order_no | 必填 | String | 30 | 商户订单号（5到30个字符、只能包含字母数字,区分大小写) |
| 07 | random_str | 必填 | String | 32 | 随机字符串 |
| 08 | sign | 必填 | String | 512 | 签名, 详见签名生成算法 |
| 09 | reserved_trade_scene | 非必填 | String | 16 | 交易场景 |

### reserved_trade_scene 交易场景值

| 场景值 | 说明 |
|--------|------|
| PARKING | 车场停车场景 |
| PARKING_SPACE | 车位停车场景 |
| GAS | 加油场景 |
| HIGHWAY | 高速场景 |
| BRIDGE | 路桥场景 |
| LIFE_PAY | 生活缴费行业 |
| K12_OFFLINE_FACE | 离线刷脸代扣场景（目前仅支持此类型） |

### reserved_debt_state 欠款状态 (K12_OFFLINE_FACE场景)

| 状态值 | 说明 |
|--------|------|
| NOT_DEBT | 无欠款（无垫资支付） |
| DEBT | 有欠款（垫资支付，有欠款） |
| REPAYMENT | 已还款（垫资支付，已还款） |

### reserved_busi_data 字段说明

| 序号 | 参数名 | 出现要求 | 型态 | 说明 |
|------|--------|----------|------|------|
| 01 | scene_info | 必填 | | 场景信息值 |
| 02 | trade_state | 必填 | String | 订单支付状态 |
| 03 | auth_appid | 非必填 | String | 车主场景必传 |
| 04 | contract_id | 非必填 | String | 签约返回的委托代扣协议id |

### trade_state 订单支付状态

| 状态值 | 说明 |
|--------|------|
| SUCCESS | 支付成功 |
| REFUND | 转入退款 |
| ACCEPTED | 已受理，未支付 |
| PAY_FAIL | 支付失败(其他原因，如银行返回失败) |

## 报文示例

**请求报文：**

```xml
<?xml version="1.0" encoding="GBK" standalone="yes"?>
<xml>
    <version>1.0</version>
    <ins_cd>08A9999999</ins_cd>
    <mchnt_cd>0002900F0370542</mchnt_cd>
    <term_id>88888888</term_id>
    <order_type>WXPAP</order_type>
    <mchnt_order_no>2020120115294222446445</mchnt_order_no>
    <random_str>随机字符串</random_str>
    <sign>签名值</sign>
    <reserved_trade_scene>K12_OFFLINE_FACE</reserved_trade_scene>
</xml>
```

**请求报文签名原文：**

```
ins_cd=08A9999999&mchnt_cd=0002900F0370542&mchnt_order_no=2020120115294222446445&order_type=WXPAP&random_str=随机字符串&reserved_trade_scene=K12_OFFLINE_FACE&sign=...
```

**响应报文：**

```xml
<?xml version="1.0" encoding="GBK" standalone="yes"?>
<xml>
    <result_code>000000</result_code>
    <result_msg>SUCCESS</result_msg>
    <ins_cd>08A9999999</ins_cd>
    <mchnt_cd>0002900F0370542</mchnt_cd>
    <random_str>随机字符串</random_str>
    <sign>签名值</sign>
    <order_type>WXPAP</order_type>
    <trans_stat>SUCCESS</trans_stat>
    <order_amt>1</order_amt>
    <transaction_id>微信订单号</transaction_id>
    <mchnt_order_no>2020120115294222446445</mchnt_order_no>
    <reserved_fy_settle_dt>20201201</reserved_fy_settle_dt>
    <reserved_fy_trace_no>富友跟踪号</reserved_fy_trace_no>
    <reserved_debt_state>NOT_DEBT</reserved_debt_state>
</xml>
```
