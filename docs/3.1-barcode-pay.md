# 3.1 条码支付（被扫）

## 请求地址

| 环境 | 地址 |
|------|------|
| 测试 | `https://fundwx.payfuiouo2o.com/micropay` |
| 生产 | `https://spay-cloud.fuioupay.com/micropay` 或 `https://spay-cloud2.fuioupay.com/micropay` |

## 场景说明

1. 用户和商户确认订单金额
2. 用户向商户展示支付二维码（微信、支付宝等）
3. 商户使用扫码枪，扫描用户二维码并提交到富友扫码

**注意**: 条码支付存在需要用户输入支付密码的场景（实时返回用户支付中），是否需要输入支付密码，与支付宝、微信的风控规则有关。此时需要商户主动发起查询来确定最终的订单状态。**建议：交易发起5秒以后开始查询，查询间隔5秒以上**。

## 请求报文

| 序号 | 参数名 | 必填 | 类型 | 长度 | 说明 |
|------|--------|------|------|------|------|
| 1 | version | 是 | String | 8 | 1.0 |
| 2 | ins_cd | 是 | String | 13 | 机构号 |
| 3 | mchnt_cd | 是 | String | 15 | 商户号 |
| 4 | term_id | 是 | String | 8 | 终端号（无真实终端统一填88888888）|
| 5 | random_str | 是 | String | 32 | 随机字符串 |
| 6 | sign | 是 | String | 512 | 签名 |
| 7 | order_type | 是 | String | 20 | 订单类型：ALIPAY/WECHAT/UNIONPAY/BESTPAY |
| 8 | mchnt_order_no | 是 | String | 30 | 商户订单号（5-30字符，字母数字）|
| 9 | order_amt | 是 | Number | 9 | 订单金额，单位：分 |
| 10 | goods_des | 是 | String | 128 | 商品描述 |
| 11 | txn_begin_ts | 是 | String | 14 | 交易开始时间yyyyMMddHHmmss |
| 12 | auth_code | 是 | String | 128 | 授权码（用户付款码）|
| 13 | term_ip | 是 | String | 40 | 终端IP |
| 14 | goods_detail | 否 | String | 1024 | 商品详情 |
| 15 | addn_inf | 否 | String | 256 | 附加数据 |
| 16 | curr_type | 否 | String | 3 | 货币类型，默认CNY |

## 响应报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | result_code | 是 | String | 错误代码，000000成功 |
| 2 | result_msg | 是 | String | 错误描述 |
| 3 | ins_cd | 是 | String | 机构号 |
| 4 | mchnt_cd | 是 | String | 商户号 |
| 5 | term_id | 否 | String | 终端号 |
| 6 | random_str | 是 | String | 随机字符串 |
| 7 | sign | 是 | String | 签名 |
| 8 | pay_msg | 是 | String | 支付状态信息 |
| 9 | transaction_id | 否 | String | 渠道订单号 |
| 10 | order_amt | 否 | String | 订单金额 |
| 11 | buyer_id | 否 | String | 买家ID |

## 返回码

| result_code | 说明 | 处理方式 |
|-------------|------|----------|
| 000000 | 成功 | 处理成功 |
| 000001 | 用户支付中 | 轮询查询订单状态 |
| 其他 | 失败 | 查看result_msg |

## 代码示例

```javascript
// 条码支付
const result = await payClient.barcodePay({
  mchntCd: '000XXXXXXX',
  orderType: 'WECHAT',
  mchntOrderNo: generateOrderNo(),
  orderAmt: 1, // 分
  goodsDes: '测试商品',
  authCode: '134XXXXXXXXX', // 用户付款码
  termIp: '127.0.0.1'
});

// 如果返回用户支付中，需要轮询
if (result.result_code === '000001') {
  // 等待5秒后开始轮询
  await new Promise(r => setTimeout(r, 5000));

  const queryResult = await payClient.queryOrder({
    mchntOrderNo: result.mchnt_order_no,
    orderType: 'WECHAT'
  });

  if (queryResult.trans_stat === 'SUCCESS') {
    console.log('支付成功');
  }
}
```
