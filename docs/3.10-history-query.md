# 3.10 历史订单查询

## 请求地址

| 环境 | 地址 |
|------|------|
| 测试 | `https://fundwx.payfuiouo2o.com/hisTradeQuery` |
| 生产 | `https://spay-cloud.fuioupay.com/hisTradeQuery` 或 `https://spay-cloud2.fuioupay.com/hisTradeQuery` |

## 场景说明

本接口提供给商户查询**历史交易记录**，支持查询90天内的交易。

查询实时交易状态须使用【3.4订单查询】。

**mchnt_order_no、channel_order_id、transaction_id 三选一**

## 请求报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | version | 是 | String | 1.0 |
| 2 | ins_cd | 是 | String | 机构号 |
| 3 | mchnt_cd | 是 | String | 商户号 |
| 4 | term_id | 是 | String | 终端号 |
| 5 | order_type | 是 | String | 订单类型 |
| 6 | trade_dt | 否 | String | 交易日期，默认当天，支持查询90天内 |
| 7 | mchnt_order_no | 条件 | String | 商户订单号（三选一）|
| 8 | channel_order_id | 条件 | String | 条码流水（三选一）|
| 9 | transaction_id | 条件 | String | 渠道订单号（三选一）|
| 10 | random_str | 是 | String | 随机字符串 |
| 11 | sign | 是 | String | 签名 |

## 订单类型

| order_type | 说明 |
|------------|------|
| ALIPAY | 支付宝 |
| WECHAT | 微信 |
| UNIONPAY | 银联 |
| WXAPP | 微信APP |
| ALIAPP | 支付宝APP |
| WXH5 | 微信H5 |
| ALIH5 | 支付宝H5 |
| WXXS | 小程序线上 |
| JDBT | 京东白条 |

## 响应报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | result_code | 是 | String | 错误代码，000000成功 |
| 2 | result_msg | 是 | String | 错误描述 |
| 3 | trans_stat | 是 | String | 交易状态 |
| 4 | ins_cd | 是 | String | 机构号 |
| 5 | mchnt_cd | 是 | String | 商户号 |
| 6 | random_str | 是 | String | 随机字符串 |
| 7 | sign | 是 | String | 签名 |
| 8 | order_amt | 是 | Number | 订单金额，单位：分 |
| 9 | transaction_id | 是 | String | 渠道订单号 |
| 10 | mchnt_order_no | 是 | String | 商户订单号 |
| 11 | buyer_id | 否 | String | 买家ID |
| 12 | reserved_fy_settle_dt | 是 | String | 富友交易日期 |
| 13 | reserved_coupon_fee | 否 | String | 优惠金额（分）|
| 14 | reserved_fy_trace_no | 是 | String | 富友系统内部追踪号 |
| 15 | reserved_channel_order_id | 否 | String | 条码流水号 |

## 交易状态 (trans_stat)

| 状态 | 说明 |
|------|------|
| SUCCESS | 支付成功 |
| REFUND | 已退款 |
| NOTPAY | 未支付 |
| CLOSED | 已关闭 |
| REVOKED | 已撤销 |
| USERPAYING | 用户支付中 |
| PAYERROR | 支付失败 |

## 代码示例

```javascript
// 查询90天前的订单（使用transaction_id）
const result = await payClient.hisQuery({
  mchntCd: '000XXXXXXX',
  orderType: 'WECHAT',
  transactionId: '4200000747202012013659872912',
  tradeDt: '20201009'
});

console.log('订单状态:', result.trans_stat);
```
