# 3.2 主扫统一下单

## 请求地址

| 环境 | 地址 |
|------|------|
| 测试 | `https://fundwx.payfuiouo2o.com/preCreate` |
| 生产 | `https://spay-cloud.fuioupay.com/preCreate` 或 `https://spay-cloud2.fuioupay.com/preCreate` |

## 场景说明

用户扫描商户展示的二维码完成支付。

## 请求报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | version | 是 | String | 版本号，1.0 |
| 2 | ins_cd | 是 | String | 机构号 |
| 3 | mchnt_cd | 是 | String | 商户号 |
| 4 | term_id | 是 | String | 终端号 |
| 5 | random_str | 是 | String | 随机字符串 |
| 6 | sign | 是 | String | 签名 |
| 7 | order_type | 是 | String | 订单类型：ALIPAY/WECHAT/UNIONPAY/BESTPAY |
| 8 | mchnt_order_no | 是 | String | 商户订单号 |
| 9 | order_amt | 是 | Number | 订单金额，单位：分 |
| 10 | goods_des | 是 | String | 商品描述 |
| 11 | txn_begin_ts | 是 | String | 交易开始时间yyyyMMddHHmmss |
| 12 | notify_url | 是 | String | 回调地址 |
| 13 | term_ip | 是 | String | 终端IP |
| 14 | goods_tag | 否 | String | 商品标记 |
| 15 | goods_detail | 否 | String | 商品详情 |
| 16 | curr_type | 否 | String | 货币类型 |
| 17 | addn_inf | 否 | String | 附加数据 |
| 18 | reserved_expire_minute | 否 | String | 订单超时时间（分钟）|

## 响应报文

| 序号 | 参数名 | 必填 | 类型 | 说明 |
|------|--------|------|------|------|
| 1 | result_code | 是 | String | 错误代码，000000成功 |
| 2 | result_msg | 是 | String | 错误描述 |
| 3 | ins_cd | 是 | String | 机构号 |
| 4 | mchnt_cd | 是 | String | 商户号 |
| 5 | random_str | 是 | String | 随机字符串 |
| 6 | sign | 是 | String | 签名 |
| 7 | qr_code | 是 | String | 二维码链接 |
| 8 | reserved_fy_order_no | 否 | String | 富友订单号 |
| 9 | reserved_fy_trace_no | 否 | String | 富友追踪号 |

## 注意事项

- 只有**主扫统一下单、公众号/服务窗统一下单**会通过回调通知结果
- 条码支付**没有**异步通知
- 建议轮询查询订单状态作为补充

## 代码示例

```javascript
// 统一下单
const result = await payClient.createOrder({
  mchntCd: '000XXXXXXX',
  orderType: 'WECHAT',
  mchntOrderNo: generateOrderNo(),
  orderAmt: 1, // 分
  goodsDes: '测试商品',
  notifyUrl: 'https://your-domain.com/callback',
  termIp: '127.0.0.1'
});

if (result.result_code === '000000') {
  console.log('二维码:', result.qr_code);
  // 展示二维码给用户扫描
}
```
